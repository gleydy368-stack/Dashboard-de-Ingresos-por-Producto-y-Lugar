# Dashboard-de-Ingresos-por-Producto-y-Lugar
Este proyecto analiza los ingresos de distintos productos (bicicletas, patines, triciclos, scooters y motos) en varias ciudades de Perú (Lima, Lurín, Trujillo y Arequipa).  

**Visualizaciones incluidas:**
- Tabla resumen de ingresos con impuestos
- Gráfico de columnas por producto y lugar
- Gráfico de barras apiladas para ingresos totales por ciudad
- Mapa geográfico de ingresos por lugar
- Gráfico de participación de productos (torta/donut)

**Objetivo:**  
Permitir identificar qué productos y lugares generan más ingresos y facilitar decisiones de ventas y logística.
